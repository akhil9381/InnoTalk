# InnoTalk

InnoTalk is a social startup market-readiness platform. The frontend now includes a real 8-phase evaluation flow, persistent local session storage, a dashboard that reflects live evaluation data, and a market-readiness report view.

## Frontend

- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run lint`

## Backend

- `cd backend`
- `npm run dev`
- `npm run lint`
- `npm test -- --runInBand --passWithNoTests`
